Overview
Inkling.Web is a lightweight, high-performance SVG editor designed to bring the power of vector graphics to your fingertips. Unlike raster editors, Inkling.Web allows for precision editing. With a focus on simplicity and workflow, it provides the essential tools needed to create stunning vector art without the bloat.

Key Features
- Drawing Tools: Smooth Brush and Eraser tools optimized for vector pathing.
- Precision Vectoring: Create paths, shapes, and lines that remain crisp at any resolution.
- Layers System: Organize complex illustrations with a robust layer management system.
- Selection Tool: Easily select, move, and transform objects.
- LocalStorage Integration: Your work is auto-saved locally, so you never lose progress.
